What is this?

This is a way to search through inspections and citations by the USDA Animal and Plant Health Inspection Service (APHIS). The data has been collected by the Data Liberation Project.

For documentation on the dataset, read this FAQ or the GitHub repository.

Data ingestion and known issues

We pull data from the repository, combine it into a single table, and use the OpenAI API to make each citation's full text searchable.

For issues with the data, see the FAQ.
